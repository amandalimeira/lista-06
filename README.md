# Lista 6 – Exploração de Estilos no React

Este repositório contém quatro implementações da mesma tela utilizando diferentes abordagens de estilização com React + Vite:

## Projetos

- **01-css-global**: Utiliza CSS global tradicional.
- **02-css-modules**: Utiliza CSS Modules para escopo local de estilos.
- **03-tailwind**: Utiliza Tailwind CSS para utility-first styling.
- **04-styled-components**: Utiliza styled-components para CSS-in-JS.

## Funcionalidades comuns

- Componentes reutilizáveis: Navbar, ProductCard, Button, Skeleton
- Dark mode com persistência
- Responsividade conforme especificação
- Tokens de design (via CSS custom properties, Tailwind config ou theme.js)

## Como rodar

Escolha o diretório do projeto desejado e siga os passos:

```bash
cd 01-css-global # ou 02-css-modules, 03-tailwind, 04-styled-components
npm install
npm run dev
```

Abra o navegador no endereço indicado pelo Vite para visualizar a aplicação.

Cada pasta possui um README.md próprio com detalhes específicos da abordagem utilizada.
